# 音楽共有サイト（プライベート版）

## 共通ルール

- やることが分からないときは **askuserquestion** でユーザに確認してください。

## 開発環境

- AWS 認証情報は **aws sso login --profile <aws-profile-name>** のコマンドで設定します。 **<aws-profile-name> は .env ファイルに記載** してあります。
- Pythonの実行は必ず `uv run` を使うこと
- パッケージのインストールは `uv add` を使うこと
- `python3` や `pip` は直接使わないこと
